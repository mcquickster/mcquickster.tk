How This Wesite Was Made

I am going to structure this as a tutorial / procedure so you can follow if you like.


  1. Install wamp server on a pc that you will use as a dedicated server.

  2. Foreward port 80 with the ip address of your pc on your router's configuration webpage.

  3. Connect the www directory to github so you can edit the files remotly and push / pull code.

  4. Install ngrok.

  5. Url foreward the nrgok url to whatever final url you want to use. (in my case mcquickster.tk from dot.tk)

  6. Write a simple powershell script to run in the background that constatly git pulls so you can update your server from any location whenever you want to.
