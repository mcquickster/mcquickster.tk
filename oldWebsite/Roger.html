Basic Stuff

HTML is a standard coding language and stands for Hyper Text Markup Language. A markup language is a language use to code a webage. This webpage is about 85% HTML. (But as you get more professional it turns to about 50%). HTML uses tags (or elements) to create something. I never said that "something" was something meaningful but whatever. Meaningful or not, HTML is a very basic coding language that is used all over the world to create simple webpages. Check out some basic tags below.

Basic Tags

First, you need the most basic tag of all: the DOCTYPE tag. This tag isn't exactly vital but it insures that the code you are writing is interpreted the right way.
Next you need the html tag. This tag shows that everything inside the html tag is html.
Inside the html tags you need to put two different tags. One is called the head tag and the other is the body tag.
Inside the head tag you put metadata about the webpage.
In the body tag you put all visible content.
