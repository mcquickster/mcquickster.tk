This page is all about javascript.

There are two ways to code with javascript. The first is to code in a seperate file for javascript (similar to a css file). The second is to code directly in a html file. In order to do this you have to use the script tag. (I can't type the script tag it into edity without turning my text to javascript.) The most basic javascript tag is alert. It simply brodcasts a message of your choice on the screen when your page loads. Click ok and it goes away. the tag looks like this: alert("type your message here").

You can use a function to call up things like a alert at a certain time such as when the user presses a button. Another thing you can do is to hide html and reveal it when the function is called. With this you can reveal secret text when a button is pressed or when the user enters a password. There are also varabels that you can use or call up if the data entered by the user matches the name. There is also if and else which you can use for basic artifical inteligence or other purposes. Overall you can do almost anything you need with javascript you just need to know how.
